Open SSH tunnel

You can open a tunnel through the SSH connection to a web endpoint/url accessible only locally on the server or on other host from the server.
After specifying the target url and clicking on Open the url will be opened in a new browser tab.

Always specify a full url with protocol (http:// or https://). If you provided a hostname not an IP and there's an issue try it with an IP address. Currently hostnames are resolved by executing a 'getent ahosts HOSTNAME' command on the ssh server that might be not available or fail there.

NOTE: this feature is currently in beta, maybe loading certain web pages fail or renders them partially broken.
